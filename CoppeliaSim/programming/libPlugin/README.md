# libPlugin: a plugin framework for CoppeliaSim

## Dependencies

Minimum versions:

- C++17 compiler (Visual C++ 2019, gcc-7, clang-3.9)
- CMake 3.16.3
- xsltproc (available via `apt`/`brew`/`choco`)
- Python 3.8
- (optional) Python package 'xmlschema' (`pip install xmlschema`)
