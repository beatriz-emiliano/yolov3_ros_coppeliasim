Regular API function

simSetObjectSpecialProperty / sim.setObjectSpecialProperty

Description Sets the special properties of a scene object. See also sim.getObjectSpecialProperty, sim.setObjectProperty and sim.setModelProperty.
C/C++
synopsis 		simInt simSetObjectSpecialProperty(simInt objectHandle,simInt prop)
C/C++
parameters
objectHandle: object handle
prop: object special property. See the object special property values. Combine them with the "or"-operator
C/C++
return value
-1 if operation was not successful
