simSetGraphStreamTransformation / sim.setGraphStreamTransformation

Description Applies a transformation to a graph stream. See also the other functions related to graphs.
C/C++
synopsis 		simInt simSetGraphStreamTransformation(simInt graphHandle,simInt streamId,simInt trType,simFloat mult,simFloat off,simInt movingAvgPeriod)
C/C++
parameters
graphHandle: handle of the graph
streamId: the id of the stream
trType: the stream transformation
mult: a multiplication factor
off: an offset
movingAvgPeriod: the moving average period. Set to 1 for no moving average
C/C++
return value
-1 if operation was not successful
