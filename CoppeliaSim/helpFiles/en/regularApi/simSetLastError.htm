simSetLastError

Description Sets a custom error or warning message and possibly raises a script error. See also simGetLastError
C/C++
synopsis 		simInt simSetLastError(const simChar* funcName,const simChar* errorMessage)
C/C++
parameters
funcName: name of the function where the message originated. Can be nullptr.
errorMessage: error or warning message. A warning message should start with warning@.
C/C++
return value
-1 if operation was not successful
