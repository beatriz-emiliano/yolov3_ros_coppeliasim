Regular API function

simExportIk / sim.exportIk

Description
DEPRECATED. Use the kinematic plugin functionality instead.
Exports the kinematics content of a scene. The generated file can be used with the Coppelia kinematics routines, to do kinematics computations in a stand-alone application.
C/C++
synopsis 		simInt simExportIk(const simChar* pathAndFilename,simInt reserved1,simVoid* reserved2)
C/C++
parameters
pathAndFilename: the location and the name of the file to create.
reserved1: reserved argument, keep at 0.
reserved2: reserved argument, keep at nullptr.
C/C++
return value
1 if operation was successful.
