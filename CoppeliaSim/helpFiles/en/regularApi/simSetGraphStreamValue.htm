Regular API function

simSetGraphStreamValue / sim.setGraphStreamValue

Description Sets the next value to be recorded for a graph stream. See also the other functions related to graphs.
C/C++
synopsis 		simInt simSetGraphStreamValue(simInt graphHandle,simInt streamId,simFloat value)
C/C++
parameters
graphHandle: handle of the graph object
streamId: the id of the stream
value: the value to set. Omitting to set a value for a corresponding sim.handleGraph call, then the data point will be missing.
C/C++
return value
-1 if operation was not successful
