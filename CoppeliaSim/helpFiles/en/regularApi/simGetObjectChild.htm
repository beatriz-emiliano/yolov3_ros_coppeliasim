Regular API function

simGetObjectChild / sim.getObjectChild

Description Retrieves the handle of an object's child object. See also sim.getObjectParent and sim.getObjectsInTree.
C/C++
synopsis 		simInt simGetObjectChild(simInt objectHandle,simInt index)
C/C++
parameters
objectHandle: handle of the object
index: zero-based index of the child's position. To retrieve all children of an object, call the function by increasing the index until the return value is -1
C/C++
return value
handle of child object or -1 if the child doesn't exist at that index or in case of an error
