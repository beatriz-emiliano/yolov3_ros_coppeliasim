Regular API function

simSetJointPosition / sim.setJointPosition

Description Sets the linear/angular position of a joint. Cannot be used with spherical joints (use sim.setObjectChildPose instead). See also sim.getJointPosition and sim.setJointTargetPosition
C/C++
synopsis 		simInt simSetJointPosition(simInt objectHandle,simFloat position)
C/C++
parameters
objectHandle: handle of the joint object
position: the linear/angular position of the joint
C/C++
return value
-1 if operation was not successful
