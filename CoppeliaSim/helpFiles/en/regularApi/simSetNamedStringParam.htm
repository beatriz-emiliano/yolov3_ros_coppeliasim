Regular API function

simSetNamedStringParam / sim.setNamedStringParam

Description Sets a named string or buffer parameter. See also the other simulator named parameter related functions.
C/C++
synopsis 		simInt simSetNamedStringParam(const simChar* paramName,const simChar* stringParam,simInt paramLength)
C/C++
parameters
paramName: the key or name of the parameter
stringParam: the string or buffer value
paramLength: the size of the string or buffer value
C/C++
return value
-1 if operation was not successful. 0 if an existing parameter was replaced, 1 if the parameter was set
