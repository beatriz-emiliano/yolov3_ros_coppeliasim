Regular API function

simSetJointDependency / sim.setJointDependency

Description Sets a joint dependent of another joint. The dependent joint should first be set into dependent mode via sim.setJointMode. See also sim.getJointDependency.
C/C++
synopsis 		simInt simSetJointDependency(simInt jointHandle,simInt masterJointHandle,simFloat offset,simFloat multCoeff)
C/C++
parameters
jointHandle: handle of the joint to become slave
masterJointHandle: handle of the joint to become master, or -1 to free the slave joint from dependency.
offset: the offset in equation slave=offset+master*multCoeff
multCoeff: the coeff in equation slave=offset+master*multCoeff
C/C++
return value
-1 if operation was not successful
