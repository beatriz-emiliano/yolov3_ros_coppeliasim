Regular API function

simRemoveBanner / sim.removeBanner

Description DEPRECATED.
C/C++
synopsis 		simInt simRemoveBanner(simInt bannerID)
C/C++
parameters
bannerID: handle of a previously added banner. sim.handle_all removes all banners from the scene. If you or-combine the bannerID with sim.handleflag_togglevisibility, then you can toggle the banner's visibility without removing it: in that case, the return value is the new visibility stae of the banner.
C/C++
return value
-1 if operation was not successful
Lua
synopsis 		sim.removeBanner(int bannerID)
Lua
parameters
bannerID: handle of a previously added banner. sim.handle_all removes all banners from the scene, that were created in a script (banners created from the C interface are not removed)
Lua
return values
