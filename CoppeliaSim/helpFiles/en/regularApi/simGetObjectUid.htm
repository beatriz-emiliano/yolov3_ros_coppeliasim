Regular API function

simGetObjectUid / sim.getObjectUid

Description Retrieves the unique identifier of an object: throughout a CoppeliaSim session, there won't be two identical unique identifiers. Unique identifiers are however not persistent (i.e. are not saved with the object). See also sim.getObjectfromUid.
C/C++
synopsis 		simInt64 simGetObjectUid(simInt objectHandle)
C/C++
parameters
objectHandle: object handle
C/C++
return value
-1 if operation was not successful
